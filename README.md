# botWhatsapp

this bot send automatically message!

so for start bot you need to install nodejs( https://nodejs.org/ ) and npm, run this command in terminal: 
npm install -g npm

than check if npm and node is installed: 
node -v
npm -v

now you can run bot, but first you need install dependences, so open repository in terminal and run: 
npm i

for starting bot type on terminal:
node index.js
